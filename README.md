# UART Bridge - Schematic & PCB Design

## 📌 Overview
This project is a UART Bridge designed using KiCad. It enables communication between two UART devices by acting as an interface module.

## ⚙️ Features
- UART to UART communication bridge  
- Designed for reliable serial data transfer  
- Compact and optimized PCB layout  
- Suitable for embedded system debugging and communication  

## 🛠️ Tools & Technologies
- KiCad (Schematic & PCB Design)  
- UART Communication Protocol  

## 🔌 Applications
- Microcontroller communication (STM32 ↔ ESP32)  
- Debugging serial interfaces  
- Embedded system development
